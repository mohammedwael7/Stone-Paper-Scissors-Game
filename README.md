# Rock Paper Scissors Game

## About The Project

A simple console-based Rock Paper Scissors game built using C++.

The player competes against the computer, where both choose one of three options:
Rock, Paper, or Scissors. The program compares the choices and determines the winner.

This project was created to practice C++ fundamentals, problem solving, and basic Object-Oriented Programming concepts.
